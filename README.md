# 🤖 Jarvis Voice Assistant

A Python-based voice assistant that performs tasks using voice commands.

## 🚀 Features

- Voice command recognition
- Open Google, YouTube, Facebook, LinkedIn
- Play songs from music library
- Fetch latest news using NewsAPI
- Text-to-speech using gTTS

## 🛠 Technologies Used

- Python
- SpeechRecognition
- gTTS
- Requests
- PyAudio

## ▶️ Run the Project

Install dependencies:

pip install -r requirements.txt

Run the assistant:

python main.py

## 🎙 Example Commands

- Open Google
- Open YouTube
- Play <song name>
- News
- Stop
